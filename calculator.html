Giant/tiny proportion calculator

About

This is a tool for calculating the proportions of a character of a given size. Note that all measurements are based on an idealized Caucasian male. This site is not under active development, but you can submit any feedback to thundering-susurrus.

Instructions

  1. Select a mode:
  2. Input the height and weight your character would be if they were a human.
  3. Input the actual height of your character.
  4. The character's computed weight will be displayed.
  5. Click between the tabs below to see sizes for specific body parts.
  6. Input the actual height of your viewer.
  7. Input the actual height of the character being observed.
  8. The observed character's perceived height from the perspective of the viewer will be displayed.
  9. Click between the tabs below to see perceived sizes for specific body parts.
