Giant proportion calculator

About

This is a tool for calculating the proportions of a giant of a given size. Note that all measurements are based on an idealized Caucasian male. This site is a work in progress!

Instructions

  1. Input how tall your giant would be if they were a human.
  2. Input the actual height of your giant.
  3. The giant's weight will be displayed.
  4. Click between the tabs below to see sizes for specific body parts.
